# Baselines

Five baseline models evaluated on the InterpretableDTIP dataset for drug-target interaction prediction.

## Environment

```bash
conda activate deepltl
```

## Data

All baselines use the **InterpretableDTIP** dataset located at `InterpretableDTIP/data/`:

```
InterpretableDTIP/data/
‚îú‚îÄ‚îÄ train/    # chem, chem.repr, protein, protein.repr, edges.pos, edges.neg
‚îú‚îÄ‚îÄ dev/
‚îú‚îÄ‚îÄ test/
‚îî‚îÄ‚îÄ params.json
```

Data format:
- `chem` ‚Äî compound IDs (one per line)
- `chem.repr` ‚Äî SMILES strings (one per line, aligned with `chem`)
- `protein` ‚Äî protein UniProt IDs
- `protein.repr` ‚Äî protein amino acid sequences
- `edges.pos` ‚Äî positive pairs: `chem,<id>,protein,<id>`
- `edges.neg` ‚Äî negative pairs: `chem,<id>,protein,<id>`

## Step 1: Data Preprocessing

Each baseline has its own preprocessing script under `Baselines/<model>/data_preprocess.py`. Run them from the project root:

```bash
# TransformerCPI ‚Äî generates pre_data/*.pkl and word2vec model
python Baselines/TransformerCPI/data_preprocess.py \
    --data_dir InterpretableDTIP/data \
    --output_dir Baselines/TransformerCPI/pre_data

# CPI_prediction ‚Äî generates pre_data/*.pkl
python Baselines/CPI_prediction/data_preprocess.py \
    --data_dir InterpretableDTIP/data \
    --output_dir Baselines/CPI_prediction/pre_data

# MPNN_CNN ‚Äî generates pre_data/*.csv (SMILES,Target,Label)
python Baselines/MPNN_CNN/data_preprocess.py \
    --data_dir InterpretableDTIP/data \
    --output_dir Baselines/MPNN_CNN/pre_data

# MSFF-DTA ‚Äî generates pre_data/*.csv + compound cache
python Baselines/MSFF_DTA/data_preprocess.py \
    --data_dir InterpretableDTIP/data \
    --output_dir Baselines/MSFF_DTA/pre_data

# AttentionSiteDTI ‚Äî uses 3D_GLAD cache directly, no separate preprocessing needed
# Requires: 3D_GLAD/cache/attention_data.pkl (see Ablations README)
```

## Step 2: Training

Each baseline has a training script in `baselines/train_<model>.py`. These scripts handle data loading, training, and save `best_model.pt` + metrics.

```bash
cd baselines

# Train all models sequentially
bash run_all.sh

# Or train individually:
python train_transformer_cpi.py          # ‚Üí Baselines/TransformerCPI/best_model.pt
python train_cpi_prediction.py           # ‚Üí Baselines/CPI_prediction/best_model.pt
python train_mpnn_cnn.py                 # ‚Üí Baselines/MPNN_CNN/best_model.pt
python train_msff_dta.py                 # ‚Üí Baselines/MSFF_DTA/best_model.pt
# AttentionSiteDTI trained via Baselines/AttentionSiteDTI/train.py
```

Training outputs are saved to `baselines/logs/<model>/`.

## Step 3: Run Inference / Generate Test Predictions

After training, run inference on the test set to generate `test_results.csv` (columns: `Label, Prediction`) for each model:

```bash
cd baselines

# Run all baselines
python run_test_csv.py

# Run specific models
python run_test_csv.py --models TransformerCPI MPNN_CNN

# Force re-run (delete existing CSVs)
python run_test_csv.py --force
```

Output files:
- `Baselines/TransformerCPI/test_results.csv`
- `Baselines/CPI_prediction/test_results.csv`
- `Baselines/MPNN_CNN/test_results.csv`
- `Baselines/MSFF_DTA/test_results.csv`
- `Baselines/AttentionSiteDTI/test_results.csv`

## Step 4: Generate Tables & Plots

```bash
cd baselines

# Generate summary table (CSV + LaTeX)
python generate_table.py
# ‚Üí results/baseline_table.csv
# ‚Üí results/baseline_table.tex

# Generate ROC and PRC curves (includes 3DGLAD)
python plot_baselines.py
# ‚Üí results/figures/roc_baselines.png
# ‚Üí results/figures/prc_baselines.png
```

The plot script also reads the 3DGLAD predictions from `results/test_predictions/3DGLAD.pkl` (generated by `test_all.py`).

## Models Summary

| Model | Drug Encoder | Protein Encoder | Decoder | Test AUC |
|-------|-------------|----------------|---------|----------|
| TransformerCPI | GCN | CNN | Transformer cross-attention | 0.9329 |
| CPI_prediction | GNN | Attention-CNN | MLP | 0.9231 |
| MPNN_CNN | MPNN (DeepPurpose) | CNN (DeepPurpose) | MLP | 0.9182 |
| MSFF_DTA | GNN | Transformer | Cross-attention | 0.9393 |
| AttentionSiteDTI | TAGConv | TAGConv + BiLSTM | Multi-head attention + MLP | 0.8862 |
| **3DGLAD (Ours)** | **GCN + GAE** | **GCN + GAE** | **Cross-attention** | **0.9430** |
